vue项目打包上线

第一步：在项目文件夹中运行npm run build 命令进行打包

第二步: 打包完成后可以看见项目文件夹众多了一个dist目录，这个里面的代码就是我们最终要上线的代码。

第三步：把 dist文件夹里的内容 给到后端同学放到后台服务器根路径下就可以了，此时访问的地址变为后端服务器的地址（在config文件夹中有配置）

如果放在后端的自己建立的一个文件夹下，例如project文件夹，需要在前端进行配置。
