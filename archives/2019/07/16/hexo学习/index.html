总结在hexo + github pages搭建中遇到的问题。

hexo d 命令中出现的问题

根据上面的错误信息提示，需要输入github的邮箱和名字：

​git config --global user.email "你的邮箱"

git config --global user.name "你的名字" ​

npm install hexo-deployer-git –save报错

npm WARN ajv-keywords@3.4.1 requires a peer of ajv@^6.9.1 but none is installed. You must install peer dependencies yourself.  + hexo-deployer-git@1.0.0 updated 1 package and audited 9369 packages in 17.172s found 132 high severi

解决办法1：手动安装ajv即可。

npm install ajv@^6.9.1

解决办法2：

先使用npm install -g npm-install-peers后再安装npm install webpack就不会报错。
